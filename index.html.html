HTML:

Hyper Text Markup Language is the standard language used to design web pages.

Heading Tags:

  • Heading tags are used to define the heading and subheadings.
  • h1 is the most important tag and h6 is the least important tag.

Paragraph:

  • p tag is defined as a paragraph tag in HTML. It is a block-level element.

Tables:

  • Tables are used to create table format in web pages.

Block-level Elements:

  • A block-level element always starts on a new line, and the browser automatically adds some space.

Inline Elements:

  • An inline element does not start on a new line and it occupies the required text space.

Formatting Tags:

  • Paragraph tag: it is defined as a paragraph
  • Bold tag: it is bold tag. This bold tag makes text bold without implying any extra importance. Ex: this is bold tag
  • Italic tag: this tag makes italic. Ex: this is italic
  • Strong tag: this tag defines text with strong importance. Ex: this is strong
  • Emphasized tag: it defines emphasized text. Ex: emphasized
  • Delete tag: it defines the text is deleted. Ex: deleted
  • Underline tag: it defines text is underlined. Ex: underline
  • Blockquote tag: it defines a section that is quoted from another source.
    Ex: this is blockquote
  • Superscript tag: it is used to render with a raised baseline. Ex: h2
  • Subscript tag: it is used to render with a lowered baseline. Ex: h2
  • Mark tag: it defines text marked or highlighted. Ex: mark
  • Abbreviation tag: this tag defines abbreviation or acronym, it provides a tooltip with the full version when hovered over. Ex: HTML
  • Preformatted tag: this tag is displayed in a fixed-width font and preserves both space and line breaks. 
    Ex: preformatted
  • Address tag: it is used to represent the address.
    Ex: address
  • Break tag: it breaks the element. It is also a self-closing tag.
    Ex: break
  • Code tag: it defines the computer code. Ex: print("hello world")
  • Keyboard tag: it defines keyboard input. Ex: input
